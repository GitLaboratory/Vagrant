Vagrant
=======

Install component's
-------------------

    $ apt-get --yes update /
    apt-get --yes upgrade /
    aptitude -y update /
    aptitude -y upgrade /
    apt-get --yes install vagrant /
    apt-get --yes install fabric

Virtual machine install script's.
---------------------------------

    $ git clone https://github.com/GitOrganization/Vagrant.git
    $ cd Vagrant
    $ vagrant up
    $ fab vagrant setup
